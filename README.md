# Real-Time Distress Sound Detection & Alert System

## Project Overview
This project aims to detect distress sounds (human screams, optionally gunshots, glass breaks, violent impacts) in real-time using AI and Python. The system provides a web dashboard with:
- Live sound level meter/bar
- Status message: "Normal" or "Alert"

## Features
- Real-time audio capture and processing
- AI-based sound classification
- Web dashboard for live monitoring

## Technologies
- Python (audio processing, AI)
- Flask (web dashboard)
- JavaScript (live audio level visualization)

## Getting Started
1. Install Python 3.8+
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open the dashboard in your browser

## Folder Structure
- `app.py`: Main backend application
- `requirements.txt`: Python dependencies
- `static/`: Frontend JS/CSS
- `templates/`: HTML dashboard
- `.github/copilot-instructions.md`: Copilot instructions
